# 开发者文档

## 项目结构

基本模块（base文件夹）
- client.py: APIClient类
- callback.py: 回调类
- crypt.py: 回调类的加解密数据算法
- exception.py: 异常类
- mixin.py: Mixin工具类

API模块：
- agent.py: 应用管理
- contact.py: 通讯录管理
- corgroup.py: 企业互联
- efficiency.py: 效率工具
- external_contact.py: 客户联系
- invoice.py: 电子发票
- media: 素材管理
- message.py: 消息推送
- oa.py: OA
- oauth2.py: 身份认证
- pay.py: 企业支付
